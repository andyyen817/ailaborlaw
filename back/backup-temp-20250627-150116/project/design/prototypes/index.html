产品概述

AI劳基法顾问是一款面向台湾用户的劳动法规AI咨询平台，通过结合AI技术与真人专家服务， 为用户提供快速、专业的劳动法规咨询，帮助用户了解自身劳动权益，解决工作中的劳动纠纷。

目标用户

  • 企业人力资源人员
  • 中小企业管理者/雇主
  • 劳工/员工

核心功能

  • AI法律咨询
  • 真人专家咨询
  • 用户账户管理
  • 推荐奖励系统

目标平台

  • Web (主平台)
  • 响应式移动端Web
  • LINE (通过二维码接入)
